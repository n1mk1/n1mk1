![OpenCV](https://img.shields.io/badge/OpenCV-5C3EE8?style=flat-square&logo=opencv&logoColor=white)

- 📐 Landmark geometry algorithms computing **X-Frame and mass ratios** (shoulders, lats, waist, quads) via center-out scans, **CLAHE enhancement, Canny edge detection**, and morphological operations
